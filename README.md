# Pod Project Frontend

A React-based frontend for the **Pod Placement System**.  
This application provides an interactive map where users can select geographic areas for pod placement, monitor pod status, and check battery levels. It supports visualization of live data from the backend to assist in disaster communication and deployment analysis.
